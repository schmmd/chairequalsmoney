Actions and Options

Next Level On Win
The game automatically proceeds to the next room when a room is completed and this option is set.

Restart Room
The room is restarted.

Next Room
Abandon the current room and proceed to the next room.

Previous Room
Abandon the current room and return to the previous room.

First Room
Abandon the current room and return to the first room.

Last Room
Abandon the current room and proceed to the last room.

Undo
Undo the last step. In other words, revert to the position before the last arrow key was pushed.

Undo Push
Undo the last push. In other words, revert to the position before the last chair was pushed.

Undo All
Undo to the begining of the room. The same as Restart Room, but with Undo All one can still redo to the state before Undo All was selected.

Redo
Redo the next step. In other words, revert to the position before the last undo.

Redo Push
Redo the next push. In other words, revert to the position after the last push was undone.

Redo All
Redo until nothing can be redone.

Replay History
Automatically replay the history of the current game.

Replay Record Solution
Automatically replay the steps of the record solution.

Stop
Stop either Replay History or Replay Record Solution.
